# BFG Melting

*Max 8.3*

Jitter Effect. Use BFGs and feedback to displace a texture.

This Jitter Effect uses two jit.bfgs with feedback to displace a texture and create a melting-effect.<br/> 
There is one control: **melting speed**<br/>
